About

Howdy, my name is Paul Trettenero, and I'm a Dallas based Full Stack Web Developer. I have a passion for building beautiful, useful web applications. I primarily use JavaScript, but picking up a new framework or language isn't a problem. I have extensive programming experience, and I'm comfortable developing on the frontend or the backend.

Backend Development: I am most comfortable working with Node.js, as well as building full MERN stack applications. I can use node.js servers to host and create all sorts of applications; full CMS websites, database-driven web applications, and even REST-driven frontend applications powered by AJAX.

Frontend Development: I dream in HTML, CSS, and JavaScript. I was trained and certified in React, but have been known flex my AngularJS muscles from time to time as well. Armed with an arsenal of CSS frameworks, I always strive to make my applications as user friendly as possible.

Mobile Development: I have the most fun when developing for mobile. I've built beautiful React Native applications, both personally and professionally. (See Overwatch Pickaxe, below)
